Реализуйте класс стека для работы со обьектами Котов (стек
строк).
Обьект Кот должен содеражать следующие поля: имя, возраст
Стек должен иметь фиксированный размер.
Реализуйте набор операций для работы со стеком:
¦ помещение котов в стек;
¦ выталкивание кота из стека;
¦ подсчет количества котов в стеке;
¦ проверку пустой ли стек;
¦ проверку полный ли стек;
¦ очистку стека;
¦ получение значения без выталкивания верхней кота из стека.
¦ сериализацию стека в json обьект и сохранение в файл
¦ чтение стека из json файл

При старте приложения нужно отобразить меню с
помощью, которого пользователь может выбрать необходимую операцию.
